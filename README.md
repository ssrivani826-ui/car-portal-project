# Car Portal Project 🚗

## 📌 Description
This is a Car Portal Web Application developed using HTML, CSS, PHP, and MySQL. It allows admin/users to manage car listings efficiently.

## 🔧 Technologies Used
- HTML
- CSS
- PHP
- MySQL

## ✨ Features
- Add Cars
- Edit Cars
- Delete Cars
- Admin Dashboard
- Responsive Design

## 🚀 How to Run
1. Download project
2. Move to htdocs (XAMPP/WAMP)
3. Import database
4. Run in browser
